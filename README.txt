Ce programme n'est pas un projet, mais juste un exemple. 
Dans cette optique, il n'a pas été prévu d'évolutions spécifiques concernant ce code. 
Cet exemple a été créé dans le but de mettre en oeuvre la lecture de fichiers audio avec SWT et JLayer via l'interface javax.sound .

Ce programme a été réalisé à mes heures perdues, il m'a fallu moins de 2 heures pour trouver les pièces du puzzle et tout coder, comme quoi ce n'est pas le bout du monde de faire un player en java si on a les bons outils ;)

L'exécutable a été réalisé avec NSIS.


Si java est installé sur la machine, l'exécutable devrait permettre de lancer l'application le plus simplement du monde. Sinon il reste le fichier bat pour trifouiller si java est mal configuré chez vous!

Ce "projet" a été créé avec Eclipse, vous pouvez le monter directement en tant que tel dans Eclipse (eclipse >= 3.0).
